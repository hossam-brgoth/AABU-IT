# (Al-Albayt University - IT College) App.

This is a simple project for IT Students, it's contain:

- Calculate Your AVG.
- Download Materials.
- View Study Plan.
- Emails of Teachers.
- Conversation Room to send messages for public.

*Developed by (Ionic Framework)
